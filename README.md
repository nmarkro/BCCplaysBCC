# BCCplaysBCC
Lua script for [Bizhawk](https://github.com/TASEmulators/BizHawk) that sets up in-game tournaments using user generated decks. Used by the [BCCplaysBCC](https://twitch.tv/BCCplaysBCC) Twitch channel.
